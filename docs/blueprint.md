# **App Name**: Unitopia

## Core Features:

- Unit Selection and Input: Enable users to select the input unit, the output unit, and enter a numeric value to convert.
- Unit Conversion and Display: Perform the unit conversion based on user input and display the result.
- Presets: Provide a list of common unit conversions for quick access.

## Style Guidelines:

- Primary color: #3498db (Blue) for a clean and professional look.
- Secondary color: #2ecc71 (Green) for success and confirmation messages.
- Accent: #f39c12 (Orange) for interactive elements and highlights.
- Clean, responsive layout that adapts to different screen sizes.
- Use clear and recognizable icons for different unit categories.
